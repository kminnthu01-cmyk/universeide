## Quick Start

```python
import asyncio
from universe import UniverseAI

async def main():
    # Create your cosmic agent ensemble
    universe = UniverseAI(
        num_agents=100,  # 100 parallel universes
        model="claude-sonnet-4-20250505",
    )
    
    await universe.initialize()
    
    # Deploy across your codebase
    result = await universe.deploy(
        task="Refactor this entire codebase for performance",
        target="src/",
    )
    
    print(result)

asyncio.run(main())
```
