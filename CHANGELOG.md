# 🪐 Universe IDE - Changelog

All notable changes to this project will be documented in this file.

## [2.1.0] - 2026-05-08

### Added
- **Intelligent Debugger** (`universe_debug2.py`)
  - SmartBreakpoint with conditions and auto-disable
  - VariableWatcher for tracking changes
  - StackAnalyzer for call stack analysis
  - ExceptionIntelligence for error analysis
  - ErrorPredictor for potential error prediction
  - DebugREPL for interactive debugging

- **AI Assistant** (`universe_ai_assist.py`)
  - AICodeAnalyzer for complexity/quality analysis
  - AIFixer for auto-fixing common errors
  - AIExplainer for code explanation
  - AIOptimizer for performance optimization
  - AIDocumenter for auto-documentation

### Fixed
- Test `test_analysis_available` now passes with fallback
